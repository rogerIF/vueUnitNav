# vueUnitNav
由vue-cli生成的前端工程，基于vue全家桶，使用element-ui 作为组件库，实现了简单的：`顶部+左侧` 联合导航。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
